Inspired by the suggestion from [xcvista to use the Pi with a PC motherboard](https://github.com/geerlingguy/raspberry-pi-pcie-devices/issues/8), Coreforge is trying to interface a Pi 5 with an AMD B350 AM4 socket, to use all the motherboard peripherals through AMD's B350 Chipset.

Yes, it's as crazy as it sounds :)

See the linked GitHub issue for more notes.
