Raspberry Pi CM5 TV Stick Lite

Website More Info
More Info GitHub Issue

Description

The Raspberry Pi CM5 TV Stick is a carrier that plugs directly to HDMI port. Perfect for Kodi, RetroPie, or just using a Pi attached directly to a monitor or TV for any use case.

Board specs:

  • Compatible with Raspberry Pi CM5 Lite and eMMC variants
  • HDMI plug
  • Micro SD card connector
  • USB-C connector for data/power
  • Two USB3.0 USB-A connectors
  • IR receiver
  • User button
  • Power button
  • RPI_nBOOT button for CM5 eMMC programming
  • Fan connector
  • Power and Act LEDs

You can find the project on @magic-blue-smoke’s GitHub here. Find the latest updates on @magic-blue-smoke’s X/Twitter.

Note: Raspberry Pi CM5 TV Stick Lite only supports CM5 modules. Older Raspberry Pi CM4 TV Stick is available on request, MOQ applies.
