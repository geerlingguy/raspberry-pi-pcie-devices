CM5 Minima

Website More Info
More Info GitHub Issue

Description

Pierluigi Colangeli designed a variety of ‘MINIMA’ CM5 carrier boards, suitable for different purposes. All are open source hardware, and can be tweaked for your own needs (indeed, someone’s already morphed one into a Cubesat build).

The three options currently available include:

  • REV1 - includes a RAK wireless LoRa module
  • REV2 - includes a 2230 M.2 NVMe slot
  • AI Edition - along with some other versions for other purposes!

You can find the REV2 on Elecrow, and the AI Edition may appear on Seeed Studios store soon.
