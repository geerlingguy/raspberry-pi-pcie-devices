MPi4 NEC MediaPlayer Kit

Website More Info
More Info GitHub Issue

Description

Sharp / NEC displays has a MPi4 NEC MediaPlayer Kit available that connects a Raspberry Pi directly into one of their display solutions using their modular display system.

Meant for digital signage applications, the board comes preinstalled with NEC MediaPlayer, and there’s a version with WiFi and an external antenna connector for wireless use.
